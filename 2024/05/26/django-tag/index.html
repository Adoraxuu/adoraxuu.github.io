Django: 從零開始學8 - 使用django-taggit 增加標籤

2024-05-26 Django

安裝Django-taggit

poetry add django-taggit

到Blog.model裡引入&使用

執行 poetry run python manage.py makemigrations blog

會出現以下結果：

執行poetry run python manage.py migrate

可以看到增加了兩張表：

使用ORM新增tag

進到shell操作看看

python manage.py shell

from blog.models import Post

到資料庫裡面就能看到剛剛新增的tag

另外一個taggit_taggeditem則是紀錄了關聯，到shell也可以查到剛剛增加的tag：

如果想要刪除則可以使用post.tags.remove(’python’) 刪除(僅刪掉關聯)

將tag加到頁面

就可以看到tags了：

也可以再加上判斷句，有tags才顯示

增加標籤連結

views新增過濾

修改網址加上tag/python，就會出現依照tag過濾的文章列表：

前台畫面加上連結

  • <a href="{% url "blog:posts_by_tag" tag.slug %}">
    • 連結可以使用{% url “app_name:path_name” 參數 %}
    • 參考project/blog/urls.py 寫的設定，app_name:path_name = blog:posts_by_tag

前台就可以看到tag有連結可以點擊：

點擊後就可以過濾顯示的文章（也就是轉換連結到/tag/tag.slug）：
